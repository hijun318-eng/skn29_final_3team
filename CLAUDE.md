## Repository skills

When `AGENTS.md` routes a task to `.agents/skills/<name>/SKILL.md`, read that file completely and follow it as the canonical reusable workflow. Keep `.agents/skills` as the single source instead of duplicating Skill files under `.claude/skills`.

## OpenWiki

이 저장소는 반복적인 저장소 문서화에 OpenWiki를 사용한다. `openwiki/quickstart.md`에서 시작해 아키텍처, 업무 흐름, 추적성과 운영 통제 문서를 확인한다.

OpenWiki는 로컬 `openai-chatgpt` provider와 승인된 ChatGPT/Codex OAuth 계정으로만 수동 갱신한다. 생성 페이지를 직접 수정하기보다 원본 code·활성 문서를 수정한 뒤 OpenWiki를 재실행하며, 생성 결과는 공식 산출물이나 구현 완료 근거로 사용하지 않는다.
