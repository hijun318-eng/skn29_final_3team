# SensePlace 데이터 표준 요약

> 2026-07-24 압축본. 상세 schema 초안은 `legacy/02_data_standard_guide_원본_20260724_1138.md`에 보관한다.

## 필수 메타데이터

- `is_synthetic`
- dataset, schema, generator version
- seed, scenario, 생성·관측 기간과 timezone
- source type, 생성 시각, 품질 검사 상태

## 데이터 구분

- 기준정보: 시설·서비스 구역·주제·지표·권한 범위
- 사실데이터: VOC, 예약·이용·대기·인력 등 합성 운영 관측치
- 업무데이터: 분석 작업, 이슈, 근거, 메모, 보고서와 결정 이력

## 품질·보안

- 필수값, 타입, 범위, 중복, 참조 무결성과 시간 정합성을 검사한다.
- 원문과 파생 분석을 분리하고 version과 식별자로 연결한다.
- 실제 고객·직원 개인정보를 생성하거나 저장하지 않는다.
- 집계 결과에는 기간·단위·표본 수를 함께 표시한다.

정확한 schema는 데이터베이스 설계 문서와 실제 migration이 생긴 뒤 확정한다.
