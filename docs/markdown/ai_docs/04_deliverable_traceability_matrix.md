- 구현·테스트 근거가 없으면 완료로 표시하지 않는다.
- 일정·담당·상태는 `../02_WBS.md`를 기준으로 한다.
- 공식 제출본과 Markdown의 내용 차이를 제출 전 확인한다.
- AI 참고 문서의 과거 경로·기술·상태를 현재 계약으로 사용하지 않는다.
- 실제 ID 매핑은 번호형 공식 문서와 테스트 보고서에서 관리한다.
