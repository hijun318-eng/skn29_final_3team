# SensePlace 호텔 VOC·운영 지원 개요

> 2026-07-24 압축본. 상세 원문은 `legacy/HOTEL_VOC_AI_AGENT_원본_20260724_1138.md`에 보관한다.

## 핵심 정의

SensePlace는 고객 VOC와 운영지표를 함께 살펴 부정 경험의 원인 후보, 근거와 점검 방향을 제시하는 호텔 운영자용 의사결정 지원 서비스다.

- 단순 리뷰 요약보다 VOC와 운영지표의 교차분석에 집중한다.
- 수치 계산과 규칙 판정은 결정론적으로 처리한다.
- LLM은 질문 해석·분류·설명에 사용하되 원인을 확정하지 않는다.
- 결과에는 기간·표본·단위·근거와 합성 데이터 표시를 포함한다.
- 실제 고객·직원·호텔 내부 데이터와 자동 조치는 범위에서 제외한다.

## 기본 흐름

`VOC 수집 → 분류·집계 → 운영지표 비교 → 원인 후보·반대 근거 → 관리자 검토 → 보고·재발 관찰`

주요 사용자는 전체 관리자와 권한이 제한된 실무 관리자다. 세부 기능·화면은 `../01_요구사항정의서.md`, `../05_화면설계서.md`와 `SensePlace_기능구체화_고도화_명세.md`를 우선한다.

## 데이터·안전 원칙

- 실데이터 확보 전 모든 업무 데이터는 `synthetic`으로 표시하고 seed와 schema version을 기록한다.
- 원문, 분석 결과, 운영지표, 조치 이력은 식별자와 version으로 추적한다.
- 개인정보·저작권·접근권한을 검토하고 최소 범위만 노출한다.
- 외부 서비스와 워커힐 내부 운영 현황은 공개 근거 없이 추정하지 않는다.

## 적용 경계

이 문서는 참고 요약이며 공식 범위·일정·구현 완료의 근거가 아니다. 최신 요구사항, WBS, 화면설계와 실제 코드·테스트가 우선한다.
