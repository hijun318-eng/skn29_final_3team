## 폴더 역할

`docs/markdown/ai_docs/`는 공식 번호 산출물과 분리한 참고 자료 보관 위치다. 다음 자료를 파일명 충돌 없이 한 단계에 모은다.

- AI가 작성한 기획·검토·계약 보조 자료
- 워커힐·호텔 VOC 관련 외부 조사와 분석
- 공식 일정·교육 운영의 조회 시점 스냅샷
- 과거 통제 문서와 저장소 구조 감사 자료

## 사용 원칙

- 현재 요구사항·범위·수용 기준은 `../01_요구사항정의서.md`를 우선한다.
- 일정·담당·상태는 `../02_WBS.md`, 화면 상세는 `../05_화면설계서.md`를 우선한다.
- 현재 구현 여부와 성능은 실제 코드·테스트·데이터에서 확인한다.
- 이 폴더의 파일명·경로·링크와 참고 작성 과정은 공식 제출본에 포함하지 않는다.
- 법령·라이선스·저작권·데이터 출처처럼 필요한 출처 표시는 제출 요건에 맞게 유지한다.
- 과거 경로와 표현이 남은 스냅샷은 현재 저장소 구조나 확정 사실로 해석하지 않는다.
