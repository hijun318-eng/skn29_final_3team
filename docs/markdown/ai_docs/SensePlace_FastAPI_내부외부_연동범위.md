# SensePlace 분석 서비스 연동 범위 요약

> 2026-07-24 압축본. FastAPI 전제의 상세 초안은 `legacy/SensePlace_FastAPI_내부외부_연동범위_원본_20260724_1138.md`에 보관한다.

## 적용 원칙

- 분석 서비스는 인증·사용자 화면·최종 업무 상태를 직접 소유하지 않는다.
- 외부 경계가 검증한 scope와 version만 받아 계산·검색·설명을 수행한다.
- 데이터 접근은 읽기 전용 허용 범위로 제한한다.
- 결과에는 근거와 계산 조건을 반환하고 관리자 결정을 대신하지 않는다.
- 브라우저가 내부 분석 endpoint를 직접 호출하지 않는다.

FastAPI 사용 여부, endpoint, worker와 배포 구조는 아직 이 문서로 확정하지 않는다. 실제 기술 결정과 구현이 생기면 코드·시스템 구성도·API 명세를 기준으로 갱신한다.
