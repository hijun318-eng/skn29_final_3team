# AI 에이전트 공용 작업 요약

> 2026-07-24 압축본. 과거 세부 가이드는 `legacy/codex_공용작업_가이드_원본_20260724_1138.md`에 보관한다.

## 작업 순서

1. 저장소 루트, 현재 branch와 `git status --short`를 확인한다.
2. `AGENTS.md`와 작업에 해당하는 Skill·기준 문서를 읽는다.
3. 현재 파일·실제 동작·관련 계약을 확인한다.
4. 가장 작은 일관된 변경을 적용하고 검증한다.
5. 개인 branch의 파일 변경은 개인 일일보고에 반영한다.

## 경계

- 요구사항은 `01`, 일정·담당은 `02`, 화면은 `05`, 구현은 코드·테스트가 우선한다.
- `ai_docs/`는 참고 자료이며 공식 범위나 구현 완료 근거가 아니다.
- `docs/templates/`는 수정하지 않는다.
- 명시적 요청 없이 stage, commit, push하지 않는다.
- 미확정 기술 스택·역할·일정을 임의로 만들지 않는다.

반복 절차는 `.agents/skills/`의 전용 Skill을 사용한다.
