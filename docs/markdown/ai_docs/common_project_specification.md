# SensePlace 공통 명세 요약

> 2026-07-24 압축본. 상세 원문은 `legacy/common_project_specification_원본_20260724_1138.md`에 보관한다.

## 공통 원칙

- 서비스는 VOC·운영지표의 근거 기반 분석과 관리자 판단 지원에 집중한다.
- 인증·권한, 분석 실행, 상태·감사 기록의 책임을 명시적으로 나눈다.
- 브라우저에 내부 분석 서비스나 데이터 저장소를 직접 노출하지 않는다.
- 계산·품질 Gate·탐지는 규칙 기반으로 재현하고 LLM 출력에는 근거를 연결한다.
- 데이터에는 `is_synthetic`, dataset·schema·generator version, seed와 기간을 표시한다.
- 보고서는 관리자 확정 전 초안 상태로 유지하며 자동 조치를 수행하지 않는다.

## 기능 축

1. 권한 범위 안에서 질문하고 표·차트·설명·근거를 확인하는 대화형 분석
2. 품질 검사를 통과한 데이터의 이상 감지, 근거 조사, 모니터링과 보고서 작성
3. 예약 연동 VOC 링크와 외부 리뷰를 통한 수집·분석 보조

기술 스택, API 수, 역할 코드와 완료 시점은 이 문서에서 고정하지 않는다. 활성 요구사항·WBS·화면설계와 실제 구현을 따른다.
