## 참고할 방향

- 직원이 자연어로 운영 데이터를 조회하는 경험
- 분산된 데이터의 용어·지표·관계를 일관되게 관리하는 방식
- 검증된 데이터만 분석에 사용하는 흐름
- 수치와 근거를 연결해 관리자 판단을 지원하는 방식

## SensePlace 적용 경계

- WISE의 내부 schema, API, 권한, VOC 기능과 실제 성과는 공개자료만으로 확정하지 않는다.
- 공개된 효율 수치와 공급사 주장을 SensePlace의 KPI나 보장값으로 전용하지 않는다.
- SensePlace는 합성 VOC·운영지표 기반의 독립 PoC로 설명한다.
- 정식 온톨로지·GraphDB 대신 versioned semantic catalog를 우선 검토한다.
- 자연어 질문은 승인된 지표·차원·권한 범위에서 처리하고 계산 근거를 제시한다.

변동 가능한 WISE·AI Guide 정보는 사용할 때 최신 공식 발표로 재확인한다.
