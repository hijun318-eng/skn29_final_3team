# 워커힐 기업분석 요약

## 확인 범위

- 워커힐은 SK네트웍스의 호텔앤리조트 사업·브랜드 단위로 분석한다.
- 서울 호텔의 숙박·예약 고객 여정과 공개된 고객 접점을 중심으로 본다.
- 객실, 다이닝, 문화·레저, 연회, 멤버십과 디지털 접점이 결합된 복합 서비스 특성이 있다.
- 공개 고객의 소리, 앱·웹 안내와 공개 AI 사례의 존재는 확인 가능하지만 내부 VOC 처리·데이터 연계 구조는 확인되지 않는다.

## 프로젝트 시사점

- 호텔·시설·여정 단계·채널·시간대를 구분해 VOC를 분석한다.
- 공개 리뷰와 프로젝트 합성 운영지표만 사용한다.
- 반복 불만, 처리 지연, 수작업 보고와 데이터 단절을 실제 현황으로 단정하지 않는다.
- 도입 타당성은 현업 수요, 데이터 제공 가능성, 담당자의 조치·재발 추적 가능성으로 별도 검증한다.

변동 가능한 기업·서비스 정보는 사용할 때 최신 공식 출처로 재확인한다.
