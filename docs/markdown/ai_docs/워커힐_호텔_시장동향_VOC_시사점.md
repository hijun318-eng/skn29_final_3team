## 시사점

- 인바운드·다국어 고객과 디지털 접점 증가는 채널·언어별 VOC 구분 필요성을 높인다.
- 복합 패키지와 시설 간 동선은 한 건의 VOC에 여러 주제가 포함될 가능성을 높인다.
- 높은 기대 수준에서는 문제 발생뿐 아니라 안내와 서비스 회복 과정도 중요하다.
- AI 분석은 개인정보 최소화, 설명 가능성, 담당자 검토와 편향 점검을 전제로 한다.
- ESG·모빌리티·시설 혼잡 이슈는 운영지표와 함께 검토하되 확인된 데이터만 사용한다.

시장 규모, 관광객, ADR·OCC·RevPAR과 공급 전망은 변동 가능한 자료다. 발표나 의사결정에 사용할 때 기준일과 최신 공식·권위 출처를 다시 확인한다.
