## 3. MVP 범위

### 반드시 구현

- 리뷰 감성 및 7개 서비스 주제 분류
- 주제별 부정 리뷰 통계와 운영 KPI 계산
- 기간별 VOC·운영지표 비교와 규칙 기반 이상징후 탐지
- 원인 후보, 근거 수치와 대응 검토 옵션 제공
- 자연어 질문 약 10개 지원
- Streamlit 대시보드
- HTML 또는 PDF 주간 리포트

### 시간 여유 시 구현

- 이슈 처리 상태 관리
- 분석 이력 저장
- LLM 기반 자유질의 확장
- 관리자 임계값 설정 화면
- 복수 호텔·지점 비교

### 제외 범위

- 실시간 호텔 시스템 연동
- 고객 대상 자동 답변 전송
- 직원 근무표 자동 변경 및 대응 방안 자동 실행
- 복잡한 인과추론
- 상용 수준의 권한·계정 관리
- 실데이터 기반 호텔 경영 의사결정

## 4. 완료 기준

- 리뷰를 감성 및 서비스 주제로 분류할 수 있다.
- 서비스별 부정 리뷰 증감률을 확인할 수 있다.
- 운영지표와 VOC를 동일 기간 기준으로 비교할 수 있다.
- 최소 3종 이상의 이상징후를 탐지할 수 있다.
- 모든 원인 후보에 근거 수치를 함께 표시할 수 있다.
- 자연어 질문 10개 중 8개 이상이 정상 동작한다.
- 조식 불만 증가 대표 시나리오를 처음부터 끝까지 시연할 수 있다.
- 주간 VOC 리포트를 HTML 또는 PDF로 생성할 수 있다.
- 새로운 환경에서 README만으로 서비스를 실행할 수 있다.
